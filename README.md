# Atlas500 Global Picks

Worldwide affiliate product-discovery storefront with verified Amazon ASIN links and CJ Dropshipping referral routes.

## Live site

https://atlas300-global-picks.eminaperla46.chatgpt.site

## Current catalog

- 35 unique Amazon product listings across 10 categories
- Every Amazon outbound URL includes the Associates tag `Atlas500-20`
- Search and category filtering
- Observable demand-signal badges where a signal was confirmed
- No copied Amazon prices or unofficial product images
- CJ Dropshipping service and registration referral links
- Shopify kept separate as a business tool until an official Impact referral URL is available
- Affiliate disclosure, privacy, terms, contact and about pages
- Public Impact site-verification text

## Local development

Requires Node.js 22.13 or newer.

```bash
npm install
npm run dev
```

## Validation and production build

```bash
npm run typecheck
npm run build
```

## Linear workflow

Connect this repository in Linear under `Settings → Features → Integrations → GitHub`. Use Linear issue IDs in branch names or pull-request titles, for example `ATL-123`, to link code changes to issues.
